## 🗃️ Basic Usage Guide

gnata works by reading JSON data and applying filters or searches with JSONata expressions. You do not need programming skills, but here are easy steps to try:

1. Prepare a JSON file on your computer. This could be a list of items, users, or any data in JSON format.  
2. Open gnata using the `.exe` file.  
3. In the command window, type a simple JSONata expression to find information. For example:  
   `$.items`  
   This command looks for all items in your JSON file.  
4. To run the command, you may need to type the file name after the expression. For example:  
   `gnata.exe yourdata.json "$.items"`  
5. Press Enter. gnata will process the file and show results quickly.
